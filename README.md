# Student Record Management System (DSA Project in C++)

## Project Overview
This is a menu-driven Student Record Management System implemented in C++ using arrays and structures.  
The main purpose of this project is to understand and implement basic Data Structures and Algorithms (DSA) concepts such as insertion, deletion, searching, sorting, and traversal.

---

## Features
- Add Student (Array Insertion)
- Display Students (Array Traversal)
- Search Student (Linear Search)
- Delete Student (Array Deletion - Shifting)
- Update Student Details
- Find Topper (Max Element in Array)
- Sort Students by Marks (Bubble Sort)
- Menu-driven Program (Loop + Switch)

---

## DSA Concepts Used

| Operation | Concept | Time Complexity |
|-----------|---------|----------------|
| Add | Array Insertion | O(1) |
| Display | Traversal | O(n) |
| Search | Linear Search | O(n) |
| Delete | Shifting | O(n) |
| Update | Search + Index Update | O(n) |
| Topper | Max Element | O(n) |
| Sort | Bubble Sort | O(n²) |
